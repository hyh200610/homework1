# 小助手网页版

一个基于 FastAPI 和 Vue 的多功能小助手应用，包含商品比价、武器刮刮乐和抽卡抽奖等功能。

## 功能特性

### 🛒 商品比价
- 支持两种商品的单价对比
- 自动计算最优购买方案
- 支持小数精度计算

### 🎮 武器刮刮乐
- 15种不同稀有度的武器卡片
- 包含剑、弓、枪三种类型
- 1-5星稀有度系统
- 已刮卡片自动记录，不重复

### 🎰 抽卡抽奖
- 支持多抽功能
- 带有保底机制（10抽保底五星，80抽保底六星）
- 支持UP角色概率提升
- 用户保底信息持久化存储

### 👤 用户系统
- 用户注册/登录功能
- 保底计数自动保存
- 数据存储于 JSON 文件

## 项目结构

## 技术栈

- **后端**: Python 3.x + FastAPI + Uvicorn
- **前端**: Vue 3 + Vite
- **数据存储**: JSON 文件

## 快速开始

### 安装依赖

```bash
# 安装后端依赖
pip install fastapi uvicorn numpy

# 安装前端依赖
cd frontend
npm install
```

### 启动服务

```bash
# 启动后端服务 (端口 8888)
python main.py

# 启动前端服务 (端口 5173)
cd frontend
npm run dev
```

### 访问地址

- 后端首页: http://localhost:8888
- 前端页面: http://localhost:5173

## API 接口

### 比价接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/bijia` | 比价表单页面 |
| POST | `/bijia_calc` | 计算比价结果 |

### 用户接口
| 方法 | 路径 | 说明 |
|------|------|------|
| POST | `/login` | 用户登录 |
| POST | `/register` | 用户注册 |
| GET | `/baodi/{username}` | 查询保底信息 |

### 刮刮乐接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/scratch/status` | 获取刮刮乐状态 |
| POST | `/scratch/play` | 刮开一张卡片 |
| POST | `/scratch/reset` | 重置刮刮乐 |

### 抽奖接口
| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/chouka` | 抽奖表单页面 |
| POST | `/chouka_do` | 执行抽奖 |
